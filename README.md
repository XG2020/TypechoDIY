# TypechoDIY
Typecho_定制美化版

##### 新更新 Typecho_1.2.0 定制美化版

美化插件来自 [gogobody](https://www.ijkxs.com) 修改自[小王先森](https://xwsir.cn) 的后台插件

## 版本特点
- 修复了头像显示问题（因国内屏蔽了Gravatar，更换为Cravatar国内源）
- 移除版本更新提示
- 修改了后台部分UI，增加了权限提示
- 增加了个性签名，登录ip显示等功能
- 后台自带美化插件，安装后手动开启
